# miceadds
#### Some Additional Multiple Imputation Functions, Especially for 'mice'

If you use `miceadds` and have suggestions for improvement or have found bugs, please email me at robitzsch@leibniz-ipn.de.
Please always provide a minimal dataset, necessary to demonstrate the problem, 
a minimal runnable code necessary to reproduce the issue, which can be run on the given dataset, and
all necessary information on the used librarys, the R version, and the OS it is run on, perhaps a sessionInfo().

#### Manual

The manual can be found here [https://alexanderrobitzsch.github.io/miceadds/](https://alexanderrobitzsch.github.io/miceadds/) 

#### CRAN version `miceadds` 3.17-44 (2024-01-08)


[![CRAN_Status_Badge](http://www.r-pkg.org/badges/version-last-release/miceadds)](https://cran.r-project.org/package=miceadds)
&#160;&#160;


The official version of `miceadds` is hosted on CRAN and can be found [here](https://cran.r-project.org/package=miceadds). 
The CRAN version can be installed from within R using:

```r
utils::install.packages("miceadds")
```

#### GitHub version `miceadds` 3.18-24 (2025-03-01)

[![](https://img.shields.io/badge/github%20version-3.18--24-orange.svg)](https://github.com/alexanderrobitzsch/miceadds)&#160;&#160;

The version hosted [here](https://github.com/alexanderrobitzsch/miceadds) is the development version of `miceadds`. 
The GitHub version can be installed from within R using:

```r
devtools::install_github("alexanderrobitzsch/miceadds")
```
